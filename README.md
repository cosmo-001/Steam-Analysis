# Project Overview

This project analyses publicly available Steam game data to explore how pricing, discounts, player engagement, and user reviews relate to a game's performance. Using Power BI, the dataset was transformed into an interactive dashboard that highlights trends across different genres and identifies factors that may influence a game's popularity.

The goal was to uncover business and product insights through data visualisation and support evidence-based decision-making.

## Objectives

- Analyse pricing and discount trends across Steam games
- Explore the relationship between price and review scores
- Identify the most popular games based on player engagement
- Compare estimated ownership across game genres
- Visualise key performance indicators using Power BI

---

## Dataset

The dataset contains publicly available Steam game information, including:

- Game name
- Release date
- Primary genre
- Price (USD)
- Discount percentage
- Review score (%)
- Total reviews
- Estimated owners
- 24-hour peak players

---

## Dashboard Features

The dashboard includes:

- Total games analysed
- Average game price
- Average review score
- Total estimated owners
- Review score distribution
- Estimated owners by genre
- Price vs review score analysis
- Discount vs popularity analysis
- Top games by 24-hour peak players

---

## Key Insights

- Game price showed little correlation with review scores, suggesting that player satisfaction is driven more by game quality than cost.
- Games with no active discount had the highest estimated ownership, likely reflecting the popularity of newly released or well-established titles.
- Early Access, Casual, and Adventure games achieved the highest average review scores, while Racing and Massively Multiplayer titles received comparatively lower ratings.
- Most games fell into the **Very Positive** or **Overwhelmingly Positive** review categories, indicating generally high levels of player satisfaction.
