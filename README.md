## Steam Player Behaviour Analysis

A BI dashboard and data analysis project examining player behaviour, pricing, 
and review sentiment across ~1,000 Steam titles (2B+ combined estimated owners).

**Dashboard includes:**
- KPI overview (avg. daily peak players, games tracked, estimated owners)
- Price vs. user satisfaction scatter plot
- Player ratings by genre
- Discount impact on game popularity
- Genre distribution by review sentiment
